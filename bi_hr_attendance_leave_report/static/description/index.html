HR Attendance and Leave Register Report Odoo App

Print Employee Attendance and Leave Report Odoo Apps

HR Employee Attendance and Leave Register Report Odoo App helps HR managers to keep track of employee attendance and leave requests. The attendance register report provides a detailed overview of employee attendance records, including selected dates and present and absences. The leave register report provides a detailed overview of employee leave records, including the number of days for allocated, utilized, and remaining leave by leave types. Users can also generate attendance and leave a register report by department and specific employee for selected dates.

Features

Employee Attendance Register Report

User can generate employee attendance register report by department or employee.

Employee Leave Register Report

User can generate employee leave register report by department or employee.

Employee Attendance Register Menu

Under Employee > Reporting menu user can see "Employee Attendance Register" menu.

On clicking menu user can see wizard, here user can generate report by department or employee, also can select "Start Date & End Date" for report.

Employee Attendance Register Report

User can see generated attendance register report for selected department.

Employee Leave Register Menu

Under Employee > Reporting menu user can see "Employee Leave Register" menu.

On clicking menu user can see wizard, Here user can generate report by department or employee, also can select "Start Date & End Date" for report.

Employee Leave Register Report

User can see generated leave register report for selected employees.
